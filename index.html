Projeto Login

Projeto construído para treinar como realizar tela de login

Saiba mais desse projeto

A construção do projeto de tela de login foi desenvolvida utilizando as linguagens HTML5 e CSS3, com foco em criar uma interface simples, funcional e visualmente agradável. O HTML5 foi responsável por estruturar os elementos da página, como campos de entrada, botões e rótulos, garantindo uma marcação semântica e acessível. Já o CSS3 foi aplicado para definir o estilo da tela, incluindo cores, tipografia, espaçamento e alinhamento dos componentes, além de ajustes responsivos para diferentes tamanhos de tela. O resultado é uma página moderna e intuitiva, que oferece uma boa experiência ao usuário
